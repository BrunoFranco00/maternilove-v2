# Ícones PWA

Esta pasta deve conter os seguintes ícones para o PWA funcionar corretamente:

## Ícones Obrigatórios

- `icon-192x192.png` - 192x192 pixels (Android)
- `icon-256x256.png` - 256x256 pixels (Android)
- `icon-384x384.png` - 384x384 pixels (Android)
- `icon-512x512.png` - 512x512 pixels (Android, Splash Screen)
- `icon-maskable-512x512.png` - 512x512 pixels (Maskable icon para Android)

## Especificações

- **Formato:** PNG
- **Background:** Transparente ou sólido
- **Maskable icon:** Deve ter padding de 20% (área segura de 80% do centro)
- **Cores:** Usar tema do app (#ec4899 ou cores da marca)

## Geração de Ícones

Você pode usar ferramentas online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://maskable.app/

Ou criar manualmente usando design tools (Figma, Sketch, etc).

## Nota

Os ícones são necessários para o PWA funcionar corretamente. 
Por enquanto, placeholders podem ser usados para desenvolvimento.
